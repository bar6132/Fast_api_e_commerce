E-Commerce Backend with FastAPI

	This project is a backend system for an e-commerce platform built using FastAPI, TortoiseORM, and other tools.


Installation:
	1. Clone the repository to your local machine
	2. Create a virtual environment
	3. install requirements by using <pip install -r requirements.txt> 


Attention !!!:
	In the Backend is needed for google 3rd party access in order to send verification emails
	To send verification emails <.env> file is required where the google credentials will be

To start FastAPI:
	Write in the console <uvicorn main:app --reload>
